# Auto-Transcribed Screen Capture for Codex

This Codex plugin teaches Codex when and how to use the `atsc` CLI for narrated macOS screen demonstrations.

It does not bundle a recorder or MCP server. Install the CLI separately so `atsc` is available on `PATH`.

```sh
npm install -g auto-transcribed-screen-capture
atsc setup-transcriber
```

For local plugin testing, add this repository marketplace to Codex or install the plugin from the Codex plugin directory when published.
